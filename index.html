Software

Conky

A free, light-weight system monitor for X on Linux and BSD, that displays any information on your desktop. Conky was originally forked from torsmo and is licensed under the GPL. Called "one of the best maintained, and definitely one of the most useful, programs in the world of open source" by Linux Magazine.

Omploader

A file hosting application I created with some help from David Shakaryan. It includes a Firefox extension I wrote which is now maintained by Jeff Glover.

Please note that I am not responsible for any of the content on Omploader, as it is public and unmoderated.

As of May 2013, I've stopped maintaining Omploader. The code is available on GitHub for those interested.
